2015-12-30 | 修饰符

const,static,extern

1. const

作用

  1. 用来修饰右边变量(基本变量,指针变量)
  2. 只要被const修饰的变量,只读

修饰基本变量:

int b = 2;
int const p ;
int const p ;
```

const - 开发中的使用场景

  1. 替换宏, 把一些常见的变量编程只读变量
  2. 方法参数, 传入一个地址, 不允许在方法内部修改外接的值

2. statiche, extern使用

static作用:

  1. 描述局部变量, 局部比那两被static修饰, 生命周期会延长(整个app运行过程中都在), 作用域不变.
  2. 局部变量被static修饰, 只会分配一次内存, 程序已启动就会分配
  3. 修饰全局变量, 全局变量被static修饰, 生命周期不变, 作用域不变, 只能在当前文件下使用

extern作用:

  1. 声明一个外部变量, 不能定义变量

其他用法

  1. static和const联合使用
    • static和const作用: 声明一个只读的静态变量
    • 开发使用场景: 在一个文件中经常使用的字符串常量,可以使用static与const组合
  2. extern和const联合使用
    • 定义全局只读变量
    • **开发规范: 全局变量不恩能够在自己的文件下定义, 有一个专门的文件, 去管理所有的全局变量 (GlobeConst.m)
    • UIKIT_EXTERN
